Detailed Description

This is the camera for 2d scenarios.

Using this class you can control all the parameters of a 2d camera. You can have as many cameras as you want. Use IND_Render::setCamera2d(IND_Camera2d *pCamera2d) method before drawing in order to choose the camera you want to use.
