Detailed Description

Camera for 3d scenarios.

Using this class you can control all the parameters of a 3d camera. You can have as many cameras as you want. Use IND_Render::setCamera3d(IND_Camera3d *pCamera2d) method before drawing in order to choose the camera you want to use.
